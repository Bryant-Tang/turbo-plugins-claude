## 修正或開發目標

> 目標編號格式為「數字 + 可選字母」，例如 `1`、`2a`、`2b`、`2c`、`3`。
> **相同數字的目標合起來必須獨立可交付**（不依賴其他數字群組的未完成目標），但每個個別目標（含字母字尾的子目標）的**範圍都要足夠小**，可在單一 chat session 的 plan mode 中完成。
> 沒有同數字子目標時，可省略字母（例如 `目標 1`）；需要同數字拆分時才使用字母（例如 `目標 2a`、`目標 2b`）。
> 若 plan mode 規劃時發現某個目標仍太大，回到 `start-dev` 將該目標拆分為更小的同數字子目標，例如原本 `目標 2a`、`目標 2b`，若 `2a` 太大可拆成新的 `2a` + `2b`，並把原本的 `2b` 重新編號為 `2c`。
